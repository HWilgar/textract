T-extract Image Tool is a web application designed to transform images into editable text. This tool uses Optical Character Recognition (OCR) technology to detect, analyze and convert images into digital text making digitizing scanned, hardcopy or handwritten document, reports or notes from different resources easier.

How to use:
  1. Simply upload images by browsing through your device or dragging them on the drop box.
  2. Click the "Convert" button.
  3. And wait for the result.
Icons:
  • - select and copy all text inside the text area.
  • - create and download a text file copy of the result
  • - set the size of the text area in full screen
  • - restore the text area size in default view
  • - remove the selected item
Want a specific part of the image?
  1. Click the uploaded image.
  2. A popup will appear, select the section of the image you want.
  3. Click "Convert".
